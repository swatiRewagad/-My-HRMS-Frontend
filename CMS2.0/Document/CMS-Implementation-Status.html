1 Project Overview

▼

CMS 2.0 is a comprehensive Complaint Management System built using a microservices architecture designed to handle end-to-end complaint lifecycle management. The system enables customers to register complaints, tracks them through configurable workflows, applies business rules for assignment and SLA monitoring, and provides officer dashboards for resolution.

Architecture

Event-driven Microservices

Services Count

12 Backend + 1 Frontend

Communication

REST + Kafka (Async)

Deployment Target

Docker / OpenShift
